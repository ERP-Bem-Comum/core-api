# Anti-Padrões Locais — clean-code-tutor

- ❌ Ensinar conceito sem citação literal de 4+ linhas.
- ❌ Atravessar 2 módulos numa resposta.
- ❌ Usar paráfrase no lugar da citação.
- ❌ Não verificar entendimento (sempre pergunte "faz sentido?").
- ❌ Confundir DRY com "não copiar código" sem citar — corrija sempre.
