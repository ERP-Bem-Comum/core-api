# Anti-Padrões Locais — clean-code-reviewer

Além dos [anti-padrões universais](../../skill-base/SKILL.md), esta skill especificamente proíbe:

- ❌ Severidade exagerada para impressionar.
- ❌ Sugerir refactoring sem mostrar antes/depois.
- ❌ Recomendar padrão GoF sem citação de Shvets.
- ❌ Revisar código de linguagem desconhecida sem sinalizar limitação.
