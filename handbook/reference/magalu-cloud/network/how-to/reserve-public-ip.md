# Como Reservar IPv4 Público

## Reservar IP Público

👣 Inicio > Menu > Network > IPs Públicos

Na página de IPs Públicos, clique no botão para + Reservar IP.

1. Confirme a reserva do IP Público
2. Você verá seu IP Público recém criado na tela de listagem de IPs

## Associar IP Público

👣 Inicio > Menu > Network > IPs Públicos

Na listagem de IPs Públicos:

1. Clique no ícone do menu "⋮"
2. Selecione "Associar IP"
3. Selecione a instância de VM à qual o IP Público será associado.
4. Confirme a ação
5. Você verá a associação recém criada na tela de listagem de IPs, coluna "Em Uso"

## Excluir IP Público

👣 Inicio > Menu > Network > IPs Públicos

Na listagem de IPs Públicos:

1. Clique no ícone do menu "⋮"
2. Selecione "Excluir"
3. Confirme a ação

> Atenção: "Só serão excluídos IPs que não estejam associados a nenhuma instância de VM."
