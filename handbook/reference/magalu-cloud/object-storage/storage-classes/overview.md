# Visão Geral

As classes de armazenamento podem ser categorizadas em duas principais categorias: uma voltada para acesso frequente e baixa latência, ideal para dados que precisam ser acessados regularmente e com rapidez. A outra é projetada para dados que são acessados raramente, oferecendo custos de armazenamento mais baixos, mas com maior latência para recuperação. A escolha entre essas classes depende da frequência de acesso e do custo desejado para o armazenamento a longo prazo.

*   **Standard**: Esta classe é voltada para acesso frequente e baixa latência, ideal para dados que precisam ser acessados regularmente e com rapidez. [Saiba Mais](standard.md)

*   **Cold Instant**: Projetada para dados que são acessados raramente, esta classe oferece custos de armazenamento mais baixos, mas com maior latência para recuperação. [Saiba Mais](cold-storage.md)
