# Consultar detalhes de um snapshot de volume

Para consultar detalhes de um snapshot execute o comando abaixo:

## Console

👣 Início > Menu > Block Storage > Meus snapshots

Ao seguir o caminho acima, serão listados os detalhes dos snapshots disponíveis.

## MGC-CLI

```bash
mgc block-storage snapshots get --id f91b0872-997f-40b3-87fa-24bbba54658b
```
