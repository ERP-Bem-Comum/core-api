# Consultar detalhes de um volume

Para consultar detalhes de um volume execute o comando abaixo:

| Nome | Tipo | Descrição | Obrigatório |
|------|------|-----------|-------------|
| id | uuid | ID do volume | Sim |

```bash
mgc block-storage volumes get --id="0377d966-4a2d-485f-81b9-37273ba26471"
```
