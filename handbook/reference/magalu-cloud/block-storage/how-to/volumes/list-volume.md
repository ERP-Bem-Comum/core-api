# Listar volumes

Para listar todos os volumes execute o comando abaixo:

## Por Block Storage

👣 _Inicio > Menu > Block Storage_

Ao seguir o caminho acima, serão listadas todos os volumes criados.

## Por Virtual Machines

Também é possível listar todos os volumes anexados a uma determinada instância. Para isso acesse o Detalhes de Instância através do produto de Virtual Machines.

👣 _Inicio > Menu > Virtual Machines > Detalhes de Instância > Volumes_

```bash
mgc block-storage volumes list
```
