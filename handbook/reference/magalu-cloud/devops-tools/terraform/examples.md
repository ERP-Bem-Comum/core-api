# Exemplos de uso do Terraform

A documentação oficial do Terraform Provider da Magalu Cloud oferece diversos exemplos práticos nos recursos disponíveis, além de guias que cobrem casos de uso simples e complexos.

Os guias e documentação mantêm-se atualizados conforme a versão do Magalu Cloud Terraform Provider utilizada.

Para acessar exemplos detalhados e referências completas, consulte a [documentação oficial do provider](https://registry.terraform.io/providers/MagaluCloud/mgc/latest/docs/).
