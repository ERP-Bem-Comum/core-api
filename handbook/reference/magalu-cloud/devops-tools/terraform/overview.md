# Visão Geral

Este guia fornece instruções passo a passo sobre como configurar e usar o provedor (_provider_) MGC no seu projeto Terraform (ou OpenTofu) e gerenciar seus recursos na nuvem Magalu.

Você pode também acessar o [registro da MGC no site da Hashicorp](https://registry.terraform.io/providers/MagaluCloud/mgc/latest/docs), onde há também a documentação referência do nosso provider.

Siga estes passos cuidadosamente para garantir uma integração bem-sucedida.
