# Visão geral

O Magalu Cloud CLI (MGC CLI) é uma ferramenta de linha de comando que permite aos usuários interagir com os serviços da Magalu Cloud (MGC) de forma rápida, eficiente e automatizada. Em vez de utilizar uma interface gráfica, os usuários podem executar comandos diretamente no terminal para gerenciar recursos, automatizar operações e integrar serviços da nuvem pública Magalu Cloud em seus fluxos de trabalho.

## Benefícios do MGC CLI

🔹 **Eficiência:** Realize operações rapidamente sem depender de interfaces gráficas.

🔹 **Automação:** Integre a Magalu Cloud em scripts e pipelines CI/CD.

🔹 **Consistência:** Execute comandos padronizados para gerenciar infraestrutura e serviços.

🔹 **Flexibilidade:** Acesse e controle recursos a partir de qualquer ambiente de linha de comando compatível.

## Funcionalidades Principais

O MGC CLI fornece comandos para gerenciar diversos serviços e recursos da Magalu Cloud, incluindo:

🔹 **Gerenciamento de Máquinas Virtuais:** Criar, listar, iniciar, parar e excluir instâncias.

🔹 **Armazenamento em Bloco:** Criar e anexar volumes de armazenamento.

🔹 **Rede e Segurança:** Configuração de grupos de segurança e redes virtuais.

🔹 **Snapshots:** Criar e restaurar snapshots para recuperação de dados.

O MGC CLI é uma ferramenta poderosa para desenvolvedores e administradores de infraestrutura que precisam gerenciar serviços da Magalu Cloud de maneira eficiente e automatizada. Com ele, é possível realizar operações essenciais via terminal, permitindo maior controle e integração com fluxos de trabalho DevOps.
