# Visão geral

As API Keys são essenciais para autenticar na Magalu Cloud, elas são versáteis e podem ser usadas em várias plataformas:

*   Interface de Linha de Comando (CLI)
*   Kit de Desenvolvimento de Software (SDK)
*   Terraform
*   Solicitações diretas de API

## Componentes de uma Chave de API

Uma chave de API consiste em três partes:

1.  **API Key**: Usada para autenticação com a API Magalu, CLI, SDK e Terraform.
2.  **Key Pair ID**: Parte do processo de autenticação para Objetos Magalu.
3.  **Key Pair Secret**: Funciona em conjunto com o ID do Par de Chaves para autenticação de Objetos Magalu.
