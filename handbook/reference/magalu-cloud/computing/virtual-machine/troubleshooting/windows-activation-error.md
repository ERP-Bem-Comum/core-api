# Suporte e solução de problemas

## Por que meu servidor aparece com uma mensagem de Licença Windows em Período de Avaliação?

Este problema ocorre quando há falha na conexão da máquina Windows com a internet durante o processo inicial de inicialização, impedindo a validação da licença.

**Solução:**

1. Acesse o painel de ativação do Windows
2. Clique em "Resolução de Problemas"
3. Aguarde a conclusão da validação do licenciamento

Após completar esses passos, o servidor funcionará normalmente e a mensagem não reaparecerá. Caso encontre dificuldades na execução do procedimento, entre em contato com o suporte técnico.
