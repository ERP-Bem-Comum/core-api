# Tipos de instâncias disponíveis

Escolha o modelo ideal para atender às necessidades da sua empresa, com a proporção equilibrada para as suas cargas de trabalho. Disponibilizamos instâncias otimizadas para diferentes cenários:

## Tipos de Instâncias

### Instâncias BV _(Balanced Value)_

As instâncias BV (Balanced Value) são máquinas virtuais que utilizam CPU compartilhada, oferecendo um excelente custo-benefício para cargas de trabalho que não requerem desempenho máximo de CPU. Estas instâncias são ideais para aplicações que não necessitam de recursos dedicados de CPU, permitindo uma melhor otimização de custos.

### Instâncias DP _(Dedicated Performance)_

As instâncias DP (Dedicated Performance) são máquinas virtuais com CPU dedicada, garantindo recursos computacionais exclusivos para sua aplicação. Estas instâncias são recomendadas para cargas de trabalho que demandam alto desempenho e consistência, como bancos de dados, aplicações críticas e workloads que necessitam de recursos computacionais garantidos.

> Instâncias **DP** estão disponíveis somente em **SE** (sudeste) no momento.
>
> Instâncias **BV** estão disponíveis em **todas** as regiões.

### GPU NVIDIA L40

As instâncias com GPU NVIDIA L40 são indicadas para workloads híbridos de IA, simulações e processamento gráfico avançado.

Elas oferecem excelente eficiência energética e ótimo desempenho em inferência de modelos pequenos e médios, incluindo LLMs de até ~70B parâmetros em quantização leve (Q4).

Ideal para quem busca equilíbrio entre performance e versatilidade em AI.

### GPU NVIDIA L40S

As instâncias com GPU NVIDIA L40S são projetadas para cargas intensivas de IA generativa, difusão e treinamentos mais pesados.

Elas oferecem desempenho até 2× superior em FP16/FP8 graças ao uso do Transformer Engine, otimizando pipelines de deep learning e modelos modernos de IA.

Ideal para quem precisa treinar modelos com maior velocidade.
