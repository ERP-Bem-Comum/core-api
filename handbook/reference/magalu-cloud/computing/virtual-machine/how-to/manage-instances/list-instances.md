# Listar instâncias

Para listar todas as instâncias execute o comando abaixo:

## Console

👣 Início > Menu > Virtual Machines

Ao seguir o caminho acima, as VMs criadas em todas as zonas da mesma região serão listadas.

## CLI

```
mgc virtual-machines instances list
```

**Info:** "O comando acima apresenta por padrão as máquinas virtuais da região Sudeste, caso queira consultar recursos que estejam na região Nordeste adicione ao comando a seguinte informação: --region="br-ne1""
