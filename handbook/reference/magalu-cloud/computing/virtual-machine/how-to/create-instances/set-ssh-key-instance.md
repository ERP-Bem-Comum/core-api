# Inserir Chaves SSH

Para adicionar uma chave SSH em uma instância, siga estas etapas:

## Console

Navegue até: 🖥️ Início > Menu > Virtual Machines > Criar Instância

Na seção Chave SSH do formulário de criação:

1. Clique em "Inserir uma chave própria nova"

2. Digite o nome da sua chave SSH pública para armazenamento. O nome pode conter apenas "letras minúsculas, hífen, underline e números"

3. Cole sua chave SSH pública no campo designado, copiando-a do seu gerador de chaves

## Informações Adicionais

Para orientações sobre geração de chaves, consulte o documento dedicado a [Gerar uma Chave SSH](gen-ssh-key-instance.md).

Documentação complementar sobre gerenciamento de chaves SSH está disponível na [seção de SSH Keys](https://docs.magalu.cloud/docs/profile/ssh-keys/overview).
