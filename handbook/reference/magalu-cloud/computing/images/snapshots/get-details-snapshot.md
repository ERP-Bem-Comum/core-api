# Consultar detalhes de um snapshot

Para consultar detalhes de um snapshot rode o comando abaixo:

## Console

👣 Início > Menu > Virtual Machines > Meus snapshots

Ao seguir o caminho acima, serão listadas todas os snapshots com detalhes.

## CLI

```
mgc virtual-machines snapshots get --id f91b0872-997f-40b3-87fa-24bbba54658b
```
