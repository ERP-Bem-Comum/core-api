OBJETIVO
────────
Gerar os 3 arquivos de secret consumidos pelo compose MySQL:
  ./secrets/mysql_root_password.txt
  ./secrets/mysql_app_password.txt
  ./secrets/mysql_readonly_password.txt

Cada arquivo:
  - conteúdo = senha em texto puro, SEM trailing newline
  - modo = 0600 (apenas owner pode ler — root no host, mysql user dentro do container)
  - encoding = utf-8

IDEMPOTENTE: não sobrescreve arquivos existentes sem --force.
TTY-AWARE: detecta stdin.isTTY e cai para --random automaticamente em CI.
ZERO DEP EXTERNA: só APIs nativas Node (ADR-0011).


ENTRADAS
────────
argv[2..]:   flags
stdin:       TTY ou redirecionado (CI)
arquivos:    presença/ausência de ./secrets/*.txt

argv aceitos:
  --random       gera senhas aleatórias 32 bytes hex (sem prompt)
  --force        sobrescreve arquivos existentes
  --help, -h     imprime usage e sai

sem flags → modo "auto":
  - stdin.isTTY === true   → prompt interativo silent
  - stdin.isTTY === false  → cai pra random com aviso em stderr


SAÍDAS
──────
stdout:
  → linhas de progresso ("✓ x criado", "↷ y já existe", "→ Pronto")
stderr:
  → avisos ("ℹ sem TTY — usando random") e erros ("Erro: flag desconhecida")
arquivos:
  → ./secrets/mysql_{root,app,readonly}_password.txt (modo 0600)

EXIT CODES (sysexits.h):
  0   EX_OK       — sucesso
  64  EX_USAGE    — flag inválida, args malformados
  74  EX_IOERR    — falha de I/O (mkdir, writeFile, chmod)


ESTRUTURA DE FUNÇÕES
─────────────────────
(top-down — função N chama funções N+1, N+2...)

main()
 ├─ parseArgs(argv)          → Args | ParseError
 ├─ printUsage(stream)       → void                (se --help)
 ├─ mkdir(SECRETS_DIR)
 ├─ resolveEffectiveMode()   → 'prompt' | 'random'  (resolve --auto)
 └─ writeSecret(name, label, args, effectiveMode) × 3
       ├─ fileExists(path)   → boolean              (idempotência)
       ├─ randomPassword()   → string               (modo random)
       └─ promptSilent(label)→ Promise<string>      (modo prompt)


PSEUDOCÓDIGO PASSO-A-PASSO
───────────────────────────

FUNCTION main():
  parsed = parseArgs(process.argv.slice(2))
  IF parsed has .error:
    stderr.write("Erro: " + parsed.error + "\n\n")
    printUsage(stderr)
    RETURN 64

  IF parsed.help:
    printUsage(stdout)
    RETURN 0

  TRY:
    mkdir(SECRETS_DIR, { recursive: true })
  CATCH:
    stderr.write("Erro ao criar SECRETS_DIR: " + err.message)
    RETURN 74

  effectiveMode = resolveEffectiveMode(parsed.mode)

  stdout.write("→ Gerando secrets em " + SECRETS_DIR + "/\n")

  FOR (name, label) IN [
    ("mysql_root_password",     "root MySQL"),
    ("mysql_app_password",      "core_app (escritor único de core.*)"),
    ("mysql_readonly_password", "readonly_bi (SELECT em core.*)"),
  ]:
    TRY:
      writeSecret(name, label, parsed, effectiveMode)
    CATCH:
      stderr.write("Erro ao gravar " + name + ": " + err.message)
      RETURN 74

  stdout.write("\n→ Pronto. Próximo: docker compose up -d mysql\n")
  stdout.write("→ NUNCA commitar /secrets/*.txt. Já está no .gitignore.\n")
  RETURN 0


FUNCTION parseArgs(raw: readonly string[]):
  mode = 'auto'
  force = false
  help = false

  FOR arg IN raw:
    MATCH arg:
      "--random"     → mode = 'random'
      "--force"      → force = true
      "--help" | "-h" → help = true
      _             → RETURN { error: "Flag desconhecida: " + arg }

  RETURN { mode, force, help }


FUNCTION resolveEffectiveMode(requested):
  IF requested === 'prompt' OR requested === 'random':
    RETURN requested        # explicit user choice; respeita

  # auto: depende de TTY
  IF process.stdin.isTTY === true:
    RETURN 'prompt'
  ELSE:
    stderr.write("ℹ stdin não-TTY detectado — usando --random automaticamente\n")
    RETURN 'random'


FUNCTION writeSecret(name, label, args, effectiveMode):
  file = path.resolve(SECRETS_DIR, name + ".txt")

  IF fileExists(file) AND NOT args.force:
    stdout.write("  ↷ " + name + ".txt já existe (use --force para regenerar)\n")
    RETURN

  IF effectiveMode === 'random':
    password = randomPassword()
  ELSE:  # 'prompt'
    password = AWAIT promptSilent(label)

  AWAIT fs.writeFile(file, password, { encoding: 'utf-8' })
  AWAIT fs.chmod(file, 0o600)
  stdout.write("  ✓ " + name + ".txt criado (chmod 0600)\n")


FUNCTION randomPassword():
  RETURN crypto.randomBytes(32).toString('hex')   # 64 chars hex


FUNCTION promptSilent(label) RETURNS Promise<string>:
  # APIs usadas: process.stdin (Node:Tty.ReadStream),
  #              setRawMode, setEncoding('utf8'), on('data')
  # Sem dep externa, sem hack `_writeToOutput`.

  RETURN new Promise((resolve, reject) => {
    process.stdout.write("Senha para " + label + ": ")

    previousRawMode = process.stdin.isRaw ?? false
    process.stdin.setRawMode(true)
    process.stdin.resume()
    process.stdin.setEncoding('utf8')

    buffer = ""

    FUNCTION cleanup():
      process.stdin.removeListener('data', onData)
      process.stdin.setRawMode(previousRawMode)
      process.stdin.pause()

    FUNCTION onData(chunk: string):
      FOR char IN chunk:
        code = char.charCodeAt(0)
        SWITCH code:
          0x0d, 0x0a:           # CR (Enter), LF
            process.stdout.write("\n")
            cleanup()
            resolve(buffer)
            RETURN
          0x03:                  # Ctrl+C
            process.stdout.write("\n")
            cleanup()
            reject(new Error("input cancelled by user (Ctrl+C)"))
            RETURN
          0x7f, 0x08:            # Backspace, BS
            IF buffer.length > 0:
              buffer = buffer.slice(0, -1)
            CONTINUE
          DEFAULT:
            # NÃO ecoa o caractere — esse é o ponto
            buffer = buffer + char

    process.stdin.on('data', onData)
  })


FUNCTION fileExists(path) RETURNS Promise<boolean>:
  TRY:
    AWAIT fs.access(path, constants.F_OK)
    RETURN true
  CATCH:
    RETURN false


FUNCTION printUsage(stream):
  stream.write(`
    Uso: pnpm secrets:setup [--random] [--force] [--help]

      --random    Gera senhas aleatórias (32 bytes hex). Sem prompt.
      --force     Sobrescreve arquivos existentes (default: skip).
      --help, -h  Mostra esta ajuda.

    Sem flags, detecta TTY automaticamente:
      - Com TTY (terminal interativo): prompt silent para cada senha
      - Sem TTY (CI, stdin redirected): cai para --random com aviso em stderr
  `)


EDGE CASES E DECISÕES
──────────────────────

1. SECRETS_DIR não existe na primeira execução:
   → mkdir({recursive: true}) cria a pasta sem erro

2. Usuário aperta Ctrl+C durante prompt:
   → reject() com mensagem; main() catch → exit 74
   → cleanup() restaura raw mode (não deixa terminal "quebrado")

3. Backspace em buffer vazio:
   → noop (não faz nada, não erra)

4. Senha vazia (user só aperta Enter):
   → aceita string vazia. Responsabilidade do user; mysql vai rejeitar
     na conexão com erro claro.

5. Senha contém aspa simples ('):
   → permitida no Node. Mas o init script bash do mysql faz interpolação
     direta da senha em SQL (heredoc). Isso é o S-1 do review W2 (escopo
     futuro). Documentar limitação.

6. Char não-imprimível digitado (ex: paste de string base64 com newlines):
   → newlines viram Enter (resolve com o que tiver no buffer). Outros
     chars de controle vão ao buffer.

7. stdin já em raw mode antes de chamar promptSilent:
   → preservamos previousRawMode e restauramos. Não assumimos "voltar pra
     non-raw".

8. Arquivo existente sem --force:
   → skip silencioso (mensagem ↷). Sem erro. Idempotente.

9. fs.writeFile com modo embutido vs writeFile + chmod separados:
   → writeFile não garante mode em todos os FS (filesystem em montagem
     pode ignorar). Chmod explícito depois é defesa em profundidade.
